Language Adapter

ukp / bert-base-multilingual-cased-ja-wiki_pfeiffer

Description

Pfeiffer Adapter trained with Masked Language Modelling on Japanese Wikipedia Articles for 250k steps and a batch size of 64.

Usage

model = AutoModel.from_pretrained("bert-base-multilingual-cased")
config = AdapterConfig.load("pfeiffer", non_linearity="gelu", reduction_factor=2)
model.load_adapter("ja/wiki@ukp", "text_lang", config=config)
